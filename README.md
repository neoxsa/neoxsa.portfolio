## ✨ Features

- 🎨 Modern dark theme with matrix-style effects
- 📱 Fully responsive design
- 🚀 Fast performance with Vite
- 📧 Contact form with EmailJS integration
- 🎭 Smooth animations and interactive UI

## 🛠️ Tech Stack

- **React 19.2.0**
- **Vite 7.2.2**
- **Tailwind CSS 4.1.17**
- **React Router DOM 7.9.5** 
- **Lucide React** - Icons
- **EmailJS** - Email service
- **Cloudflare Turnstile** - Turnstile integration
- **Sonner** - Toast notifications
## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/neoxsa/portfolio_sa.git
   cd portfolio_sa
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open** `http://localhost:5173` in your browser

## 🎨 Customization

Edit `src/constants/constants.js` to add your projects:

```javascript
const all_projects = [
  {
    id: 1,
    title: 'Your Project',
    description: 'Description',
    image: your_image,
    tech_stack: ['React', 'Tailwind CSS'],
    category: 'Frontend',
    liveLink: 'https://your-link.com',
    repoLink: 'https://github.com/yourusername/repo'
  }
]
```
